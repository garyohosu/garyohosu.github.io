# Fable 5初心者向けnote改稿設計

## 目的

`drafts/fable-5-note.md` の話題紹介としての読みやすさを維持しながら、初心者が概要、利用条件、試し方、注意点を誤解なく把握できる記事にする。

## 編集方針

- Anthropic公式情報を、モデル概要、提供再開の経緯、対象プラン、安全対策の根拠にする。
- X由来の作例は、第三者記事で確認できたMinecraft風・ポケモン風ゲームに限定する。
- 作例は完成品や再現性を保証するものではなく、投稿・報道に基づく事例だと明記する。
- 確認できなかったSpider-Man風ゲームとWebサイトのアプリ変換例は削除する。
- Claude.aiで試す基本手順と、初心者がコピーできる小規模なプロンプト例を追加する。
- AI生成物の動作、権利、安全性は利用者が確認すべきことを明記する。
- note投稿用の内部メモは本文から削除する。

## 確認方法

- 本文中のリンクがHTTPSであることを確認する。
- MarkdownとYAMLフロントマターを目視確認する。
- リポジトリのローカルビルドを実行する。
- push後に `node scripts/ai-post-push-check.mjs` を実行する。
