# Web アプリアイデア集（X ウケ・派手・驚き要素）

## 1. 🎯 AI 業界人物相関図ジェネレーター
### コンセプト
- OpenAI, Google, Meta, Anthropic など、AI 業界の主要人物（CEO/CTO/研究者）の相関図を **インタラクティブに表示**
- ノードをドラッグで移動、クリックで経歴・つながりを表示
- AI 進化のタイムライン（2017–2025）でアニメーション

### 派手さポイント
- 📊 Force-directed graph で、人物同士の「引力」が可視化
- ✨ ホバーで人物の顔画像 + 実績が浮き出す
- 🎬 タイムラプス再生で「AI 業界の勢力図の変化」を動画化
- 🌐 各人物をクリック → Wikipedia / X / 公式ページへリンク

### 技術スタック
- Three.js / Babylon.js（3D 相関図版）
- D3.js（2D 版）
- Canvas アニメーション

---

## 2. 🎲 LLM トーナメント（リアルタイムバトル）
### コンセプト
- **同じプロンプトを複数の LLM に投げて、リアルタイムで「勝ち負け」を判定**
  - ChatGPT vs Claude vs Gemini vs Llama
  - 「1000 単語のエッセイ」「コード生成」「創作」など複数カテゴリ

### 派手さポイント
- ⚡ **リアルタイムで各 LLM の生成文字が流れてくる**（タイピング風アニメ）
- 🏆 進捗バーで「どの LLM が速いか」を可視化
- ⭐ 最終判定画面で大きく「WINNER」の表示
- 🎬 マウスオーバーでモデルサイズ・パラメータ数を表示
- 📊 グラフで「カテゴリ別の勝率」を表示

### 技術スタック
- 複数 LLM API の並列呼び出し（OpenAI, Anthropic, Google など）
- WebSocket で リアルタイム更新
- Canvas / SVG でアニメーション

## 3. 🌊 AI ニュースの「波」ビジュアライザー
### コンセプト
- RSS/API から取得した AI ニュースを **波のアニメーションで表現**
- 大きなニュース = 大きな波
- リアルタイム更新で波が湧き出ていく

### 派手さポイント
- 🌊 **波の大きさで「ニュースの重要度」を表現**（クリックで記事表示）
- 🎨 カラーで「カテゴリ」を表現（赤=規制, 青=技術, 緑=企業 など）
- 💫 波同士が「衝突」する animation
- 🔊 音声フィードバック（低い音で大きなニュース）
- ⏱️ 時系列で波を「再生・巻き戻し」可能

### 技術スタック
- Three.js（3D 波のシミュレーション）
- Web Audio API（音声生成）
- リアルタイムデータソース（HN, Product Hunt など）

---

## 4. 📈 AI モデル進化のパーティクルエフェクト
### コンセプト
- GPT-2, GPT-3, GPT-4, GPT-4o などのモデル進化を **パーティクル爆発で可視化**
- 各モデルの性能指標（精度・速度・コスト）を X, Y, Z 軸で配置
- モデル登場時に「爆発」アニメーション

### 派手さポイント
- 💥 **新モデル発表で画面全体が爆発**（confetti + particle effect）
- 🎆 モデル同士の「距離」が性能差を表現
- 🌈 グラデーションカラーで性能進化を可視化
- 📊 スライダーで時間軸を移動、過去のモデルとの比較
- 🎬 "シミュレーション" で「2026年のモデル」の予測位置を表示

### 技術スタック
- Three.js（3D パーティクル）
- Canvas（2D 版）
- データは AI モデルベンチマーク（MMLU, MT-Bench など）から取得

## 5. 🎬 AI 生成動画 × インタラクティブ・ストーリーテリング
### コンセプト
- **Sora / Runway などで生成された動画の「分岐ストーリー」**
- ユーザーが選択肢をクリックすると、**リアルタイムで次の動画が生成される**（デモとしてプリレンダリング）

### 派手さポイント
- 🎥 **Full HD/4K 動画が画面いっぱいに再生**
- 🎮 選択肢クリック → 動画切り替え + トランジション effect
- ✨ 動画中に「テキスト/ UI要素」がマスキング・アニメーション
- 🎭 複数エンディングで遊び心
- 📱 モバイル対応で "フルスクリーン immersive 体験"

### 技術スタック
- HTML5 Video
- Framer Motion / GSAP（トランジション）
- WebGL（動画フィルター・エフェクト）

## 6. 🎪 「AI がクイズ出題」＆ リアルタイムスコアランキング
### コンセプト
- **ChatGPT/Claude が出題するクイズに答える**
- 難易度は「前回の成績」に基づいて動的調整（IRT: Item Response Theory）
- **全ユーザーのスコアがリアルタイムランキング表示**

### 派手さポイント
- 🏆 **ランキング画面が大きく表示**、自分の順位に矢印が指す
- 💫 正解で「爆発 + 音 + スコア加算」
- 🌍 世界地図上に「今この瞬間に解いてる人」を表示
- 📊 自分のスコア遷移グラフがリアルタイム更新
- ⏱️ 「難易度別」「カテゴリ別」でタブ切り替え

### 技術スタック
- WebSocket（リアルタイムランキング）
- LLM API（クイズ生成）
- Firebase / Supabase（スコア保存）
- D3.js（グラフ）

## 7. 🌀 「AI 業界の給与・職位マップ」インタラクティブ可視化
### コンセプト
- **AIエンジニア・研究者の給与分布を、企業 × 職位 × 経験年数で可視化**
- バブルチャートで「雲のように」表現
- フィルタで「YC スタートアップ」「GAFAM」など条件指定

### 派手さポイント
- 💰 **バブルのサイズ = 給与**、クリックで詳細情報
- 🌈 色で「企業分類」を表現
- 🎯 マウスホバーで「平均給与」「求人数」を表示
- 📈 時間軸で「給与の推移」をアニメーション
- 🗺️ 地図上に「企業の本社 + 平均給与」を表示

### 技術スタック
- D3.js / Plotly（バブルチャート）
- Mapbox（地図）
- リアルタイムデータ源：Levels.fyi, Blind など

## 8. 🎨 「AI アートジェネレーター ＆ NFT モックアップ」

- 🖼️ 3D ギャラリー空間で「絵画を360°回転」
- ✨ 選定した画像が「額縁に入ってフレーム化」
- 💎 シェアボタンで OpenSea モックアップメタデータを表示
- 🌈 色彩の「パレット分析」をリアルタイム表示

### 技術スタック
- Three.js（3D ギャラリー）
- DALL-E / Stable Diffusion API
- Canvas（パレット分析）
