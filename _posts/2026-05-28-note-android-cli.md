## Android CLIって何？（中学生でもわかる説明）

**CLI（シーエルアイ）** とは、「Command Line Interface（コマンドライン インターフェース）」の略です。

要するに、**キーボードで「これやって！」と命令を打つと、コンピュータがその通りに動いてくれる仕組み**です。

マウスでポチポチクリックする代わりに、文字で指示を出す感じですね。

Googleが作ったこの「Android CLI」は、特に**AIの相棒（Claude CodeなどのAIコーディングツール）** がAndroidアプリを作る時に大活躍するツールです。

### なぜすごいの？

- AIがアプリを作る時に、普段はたくさんの説明（トークン）を必要とする
- このツールを使うと、**短い命令で正確にアプリを作れる**ようになる
- 結果として、**お金（API料金）が節約**できて、**速く正確に**アプリが完成する

例えるなら、**優秀な工場長に「この部品を組み立てて」って一言で指示が出せる**ようなもの。工場長（AI）が勝手にいい感じに作ってくれます。

## 記事の中で実際に試したこと

1. **インストールは超簡単**
   - Windowsなら `winget install --id Google.AndroidCLI` という1行の命令だけ
   - MacやLinuxも似たような簡単コマンドでOK

2. **AIにアプリを作ってもらう**
   - Claude Code（AIのプログラマーさん）に「Android CLIを使ってカウンターアプリを作って」とお願い
   - すると、**+ボタンで数字が増える、-ボタンで減る**シンプルなアプリが自動で完成
   - 色が変わったり、ちゃんと動く本物のAndroidアプリが出来上がる

記事には実際に作ったアプリのスクリーンショットや、完成したコードの説明も載っています。
