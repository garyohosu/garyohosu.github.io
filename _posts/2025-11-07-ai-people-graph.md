## AI 業界の「人脈ネットワーク」を可視化したアプリを公開しました

AI 業界の構造を理解するには、**どの人物がどう繋がっているか** を知ることが重要です。

新しく公開したアプリ「**AI 業界人物相関図ジェネレーター**」では、Sam Altman, Demis Hassabis, Yann LeCun など **AI 業界の主要人物と、その関連性を可視化** できます。

## 主な機能

### 1. 🌐 Force-directed グラフ（力学モデル）
- 人物をノードとして表示
- 関連性のある人物が「引き合う」アニメーション
- ドラッグで移動可能（自由に探索）

### 2. 🔍 検索機能
- 人物名で検索すると、該当者が強調表示
- 他の人物は薄くなり、フォーカスが明確

### 3. 🏢 企業フィルタ
- OpenAI / Google / Anthropic など企業ごとに表示
- 「どの企業に誰がいるか」を一目で把握

### 4. 📅 タイムラインスライダー
- 2017–2025 年の進化を追跡
- 「2020 年時点では誰がいたか」を確認可能

### 5. 📌 詳細情報パネル
- ノードをクリックすると、右側パネルに詳細表示
- 職位、企業、参加年、簡潔な経歴

### 6. 📊 統計情報
- 現在表示中の人物数
- 関連性（リンク）の総数

## どう使うか

### 初心者向け

```
1. ページを開く
2. グラフを眺める → AI業界の全体像が分かる
3. ノードをクリック → 人物の詳細情報が表示
```

### 深掘り向け

```
1. 「OpenAI」フィルタで企業を絞る
2. タイムラインで「2015 年」にセット（OpenAI創設時）
3. Sam Altman と Ilya Sutskever の関連性を確認
4. 他企業の成長と比較
```

### 業界研究向け

```
1. Yann LeCun で検索 → Meta の AI 戦略が可視化
2. Geoffrey Hinton で検索 → Turing 賞受賞者の系図が明らかに
3. 2020 年 → 2025 年 のタイムラインで業界の転換点を把握
```

## なぜこのアプリを作ったのか

**AI 業界は、『個人の人脈』で成り立っている** ことが多いです。

- Sam Altman は Y Combinator から OpenAI へ
- Elon Musk は OpenAI 共同創設者
- Demis Hassabis は DeepMind → Google へ

こうした背景を知ると、**ニュース理解が一段深くなります**。例えば：

- 「OpenAI が AWS と提携した」→ Satya Nadella の戦略の一環
- 「Meta が AI に投資」→ Yann LeCun の影響力
- 「Google DeepMind が AlphaFold」→ Demis Hassabis のビジョン

## 技術スタック

- **D3.js v7**：力学モデルのグラフビジュアライゼーション
- **HTML5 Canvas**：ノード・リンクのレンダリング
- **バニラ JavaScript**：インタラクション
- **CSS3**：グラデーション・アニメーション

## 🚀 v2 の新機能（2025/11/07 アップデート）

### 1. 人物数を大幅拡張（32人以上）

**新規追加：**
- **起業家・投資家層**：Kai-Fu Lee (Sinovation), Oren Etzioni (AI2)
- **研究者層**：Ian Goodfellow (GAN発明者), Fei-Fei Li (ImageNet), Pieter Abbeel (RL先駆者)
- **アカデミア**：Silvio Savarese, Sergey Levine
- **業界リーダー**：Jensen Huang (Nvidia), Bill Dally
- **歴史的人物**：Geoffrey Hinton (Backprop発明者), Yoshua Bengio (DL三巨頭)

### 2. 共著論文の可視化

- 論文の共著者関係を **ピンク色の破線** で表示
- 異なる企業間の学術的つながりが一目瞭然
- 「Ilya Sutskever と Chris Olah」「Yann LeCun と Yoshua Bengio」など

### 3. 職位別フィルタ

4つのボタンで役職を選択可能：
- **CEO**：経営者層（大きなノード）
- **CTO/VP**：技術責任者（中サイズ）
- **研究者**：学術研究者（小サイズ）
- **その他**：投資家、顧問など

複数選択も可能。例：「CEO と CTO だけ見たい」

### 4. タイムラインスライダー（2010–2025）

年代ごとに人物を追跡：
- 「2015 年時点では OpenAI に誰がいたか」
- 「2013 年から Google は何人の人物を獲得したか」
- AI 業界の歴史的な人員配置の推移を動画で観察

### 5. ノードのクリック詳細化

選択した人物について表示：
- **基本情報**：職位、企業、参加年
- **人物説明**：簡潔な経歴・貢献
- **同企業の人物**：同じ企業で働いた/いる人
- **共著論文**：一緒に論文を書いた研究者

### 6. ビジュアルの派手さ強化

- **ノードサイズ**：CEO は大きく、強調表示
- **ノード色**：職位で色分け（CEO=ピンク、CTO=青、研究者=水色など）
- **グロー効果**：ホバーで ピンク色に輝く
- **選択時**：黄色の枠線で特別な強調
- **リンク色**：同企業=青色、共著=ピンク色
